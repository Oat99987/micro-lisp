# micro-lisp
A Lisp interpreter written in C. Under development. 

## Currently:
a) Parser
b) Basic Evaluation

## IN PROGRESS:
a) Better error handling (for example, `clisp> / 10 0` returns errors).
b) The interpreter in general.

Please note that this will be a fairly minimal adaptation of Lisp.

**MPC (Micro Parser Combinator) Library:** https://github.com/orangeduck/mpc

---

This interpreter was built following the book "Build Your Own Lisp" available at(https://www.buildyourownlisp.com).
