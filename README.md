# micro-lisp
Current version: 0.0.0.3





A Lisp interpreter written in C. 

## Currently:
a) Parser
b) Evaluator
c) Error Handling

## IN PROGRESS:
A)The interpreter in general.
b) S-Expressions, Q-Expressions etc.
Please note that this will be a fairly minimal adaptation of Lisp.

**MPC (Micro Parser Combinator) Library:** https://github.com/orangeduck/mpc

---

## Installation
``` Bash
git clone https://www.github.com/Oat99987/micro-lisp.git
cd micro-lisp
make clisp
./clisp
```
And there you have it, A minimal lisp interpeter in a REPL environment. Please note that I'm still working on stuff like S-expressions and Everything else basically.

S-expression support may come later today!




This interpreter was built following the book "Build Your Own Lisp" available at(https://www.buildyourownlisp.com).
